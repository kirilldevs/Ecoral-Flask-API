API Documentation: Analyze Social Media Posts

Endpoint: /analyze-posts

Method: POST

Description: This endpoint analyzes social media posts related to diving and marine life, extracting information such as animals, environment, relevance to diving, location, number of people, and time of day from both text and images.

Input

Content-Type: application/json

Body:

{
  "arr": [
    {
      "text": "Example post text about diving",
      "image": "URL to an image related to the post"
    }
  ]
}

Success Response

{
  "data": [
    {
      "animals": "Detected animals",
      "environment": "Detected environment",
      "isAboutDiving": true or false,
      "location": "Detected location",
      "number of people": "Detected number of people or null",
      "time of the day": "Detected time of the day"
    }
  ],
  "message": "Data processed",
  "status": "success"
}

Error Response

Invalid Input (400 BAD REQUEST):

{
  "message": "Invalid input format",
  "status": "error"
}

Processing Error (500 INTERNAL SERVER ERROR):

{
  "message": "Error processing data",
  "status": "error"
}

Sample Call

curl -X POST https://api.yourdomain.com/v1/analyze-posts      -H "Content-Type: application/json"      -d '{"arr":[{"text":"Example text","image":"Image URL"}]}'

Notes:

Text analysis is performed by the Gemini NLP model. Image analysis is conducted using a custom-built neural network model.
