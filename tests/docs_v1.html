Upload Endpoint

The upload endpoint allows for the uploading of text and images for processing.

URL

/api/v1/upload

Method

POST

Content-Type

application/json

Body Parameters

The request body must include a JSON object with a "data" key containing nested "text" and "image" keys.

{ "data": { "text": "Example text", "image": "https://example.com/image.jpg" } }

Response

The response will be a JSON object containing the processed data.

{ "status": "success", "message": "Data processed", "text": "Example text", "image": "https://example.com/image.jpg", "answer": "Processed data" }

Ensure your requests are correctly formatted to receive the desired response.
